
# Floventry

Inventory and operations management system.

## Overview
- Purpose: Manage inventory, warehouses, and company operations in one platform.
- Maintainer: Solo-developed and actively maintained.

## Status
- Active development.
- Intended for internal use and iterative improvement.

## Tech Stack
- Backend: NestJS (TypeScript)
- Frontend: Next.js (TypeScript), Tailwind CSS

## Project Structure
- [backend](backend): API, business logic, GraphQL schema
- [frontend](frontend): Next.js application and UI

## Quick Start
1. Install workspace dependencies:

```bash
pnpm -w install
```

2. Start backend:

```bash
cd backend
pnpm run start:dev
```

3. Start frontend:

```bash
cd frontend
pnpm run dev
```

## Environment
- Create environment files for both packages (for example, `.env.example` to `.env`).
- Configure required values such as API URL, database connection, and authentication keys.
